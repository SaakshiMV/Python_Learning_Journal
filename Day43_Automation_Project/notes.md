# 📝 Day 43 — Automation Project (Email Sender)

## Overview

Welcome to **Day 43 of my 45-Day Python Learning Journey**.

Today I built an **automated email sender using Python**, a practical tool for automating communication tasks.

---

## 🚀 What It Does

- 📄 Reads email list from a file  
- 📧 Sends emails automatically  
- 👥 Supports multiple recipients  

---

## 📌 Use Cases

- 🔔 Sending notifications  
- 📢 Bulk emails  
- ⏰ Reminders  

---

## 🧠 What I Learned

- Working with `smtplib` (SMTP)  
- Sending emails using Python  
- Handling secure login (email + app passwords)  
- Automating communication workflows  

---

## ⚙️ How It Works

1. Load recipient emails from a file  
2. Connect to SMTP server  
3. Login securely  
4. Send emails to all recipients  

---

## ▶️ How to Run

```bash
python email_sender.py
````

---

## ⚠️ Important Note

* Use **App Passwords** instead of your main email password
* Avoid exposing credentials in code
* Use environment variables for security

---

## 💭 Reflection

This project demonstrates how Python can automate **real-world communication tasks**.

It opens the door to building systems for **notifications, alerts, and scalable messaging** 🚀
