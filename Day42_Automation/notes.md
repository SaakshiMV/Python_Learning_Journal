## 🚀 What I Built

A Python script that:

- 📂 Scans a folder  
- 🏷️ Identifies file types  
- 📁 Organizes files into folders automatically  

---

## 🔄 Example

### Before:
```

folder → random files

```

### After:
```

Images/
Documents/
Videos/
Others/

````

---

## 🧠 What I Learned

- Working with the `os` module  
- Automating file handling  
- Writing scripts for real-world use  
- Improving productivity through automation  

---

## ⚙️ How It Works

1. Scan all files in a directory  
2. Check file extensions  
3. Create folders if they don’t exist  
4. Move files into categorized folders  

---

## ▶️ How to Run

```bash
python file_organizer.py
````

* Cleaning up messy directories
