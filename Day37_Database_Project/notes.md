## 🗂️ Project Structure

```
expense-manager/
│
├── database.py     # Database connection & queries
├── models.py       # Expense data structure
├── services.py     # Business logic
├── main.py         # CLI / application entry point
└── expenses.db     # SQLite database
```

---

## 🧠 Core Functionalities

### 1️⃣ Add Expense

- Inserts a new record into the database  
- Includes title, amount, and category  

---

### 2️⃣ View Expenses

- Fetches all records from the database  
- Displays them in a clean format  

---

### 3️⃣ Update Expense

- Modifies existing entries using `id`  
- Updates fields like title, amount, or category  

---

### 4️⃣ Delete Expense

- Removes a record based on its unique ID  

---

### 5️⃣ Filter by Category

- Retrieves only expenses matching a specific category  

---

### 6️⃣ Total Expenses

- Calculates total spending using SQL aggregation
