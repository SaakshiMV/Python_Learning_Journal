# 📝 Day 41 — Flask Mini Project (Expense Tracker Web App)

## Overview

Welcome to **Day 41 of my 45-Day Python Learning Journey**.

Today I built a **full-stack Expense Tracker Web App** using Flask and SQLite.

This project combines **frontend, backend, and database** into one complete application.

---

## 🚀 Features

- ➕ Add expenses using a form  
- 📋 View all expenses  
- 💾 Store data in SQLite database  
- 🎨 Simple UI with HTML + CSS  

---

## 🧰 Tech Stack

- 🐍 Python (Flask)  
- 🗄️ SQLite  
- 🌐 HTML + CSS  

---

## 🧠 What I Learned

- Connecting Flask with a database  
- Handling form data in a real-world project  
- Structuring a full-stack application  
- Managing data flow between frontend and backend  

---

## ⚙️ How It Works

1. User submits expense via form  
2. Flask receives the request  
3. Data is stored in SQLite database  
4. Expenses are retrieved and displayed on the UI  

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
````

Open in browser:

```
http://127.0.0.1:5000/
```
