## 🚀 Features

- ➕ Add expenses  
- 📋 View all expenses

- 🏷️ Categorize expenses  
- 💰 View total spending

- 🗄️ SQLite  
- 🌐 HTML + CSS

## ⚙️ How It Works

1. User interacts with the web interface  
2. Flask handles requests and processes data  
3. Data is stored and retrieved from SQLite  
4. Results are displayed dynamically
